Pasos para poder migrar y seedear la base de datos.

1-  Primero sera ubicarnos en la carpeta de "backend" en la terminal de VisualStudio o la de tu sistema operativo

2-  Una vez ubicados en la carpeta "backend" pondremos este comando "python start.py".
    Dejamos que lo haga todo y ya estaria.
